1.Vision
SAKURA is a modular AI Operating System designed to provide long-term memory, intelligent planning, tool execution, knowledge retrieval, and personalized assistance through a scalable cognitive architecture.

Unlike traditional chatbot applications, SAKURA separates reasoning, memory, planning, tool usage, and language generation into independent modules, allowing each subsystem to evolve independently.
